# Computer control train file
# Format: trigger => ACTION|arg1|arg2||ACTION2|arg1|arg2
# Rule match: prompt phai bat dau bang trigger (khong phan biet hoa/thuong, co/khong dau)
# Placeholder ho tro trong arg:
#   {PROMPT}      : toan bo prompt goc
#   {PROMPT_CLEAN}: prompt da chuan hoa khong dau
#   {REST}        : phan prompt con lai sau trigger
#   {NOW}         : thoi gian hien tai yyyy-mm-dd HH:MM:SS
#   {NL}          : ky tu xuong dong
#
# Action ho tro:
# OPEN_FILE/READ_FILE|path
# LIST_DIR|path
# WRITE_FILE|path|content
# APPEND_FILE|path|content
# CREATE_FILE|path|optional_content
# CREATE_DIR|path
# DELETE_FILE|path
# DELETE_DIR|path
# DELETE_PATH|path
# SCAN_DESKTOP_CLOCK
